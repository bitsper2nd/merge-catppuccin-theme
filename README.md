# Theme - Catppuccin Nova

[Catppuccin](https://catppuccin.com/) theme for Sublime Merge 2.

This implementation of Catppuccin uses the Sublime Merge Theme as a template with the Catppuccin color palette.

## Install

Assuming the required color scheme is installed, git clone the project or download it into your Sublime Merge `Packages`
folder as `Catppuccin Theme`. Edit your merge `Preferences.sublime-settings` file to use the Catppuccin theme:

```
    "theme": "Catppuccin Nova (Latte).sublime-merge-theme"
```
